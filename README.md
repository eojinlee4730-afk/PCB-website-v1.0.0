# SUNGHWA HITECH Website

## 1. Overview

This project is a corporate website for SUNGHWA HITECH, focused on showcasing PCB products and company information.

* Purpose: Portfolio + real-world business website
* Tech Stack: HTML, CSS, JavaScript
* Deployment: Vercel
* Domain: https://sunghwahitech.com

---

## 2. Features

* Company introduction page
* Product gallery (PCB types)
* Responsive design (mobile/desktop)
* SEO optimization (meta tags, sitemap, robots.txt)
